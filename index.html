Projects

02

3D Weather App

Crafted a visually striking 3D Weather App that transforms raw weather data into an immersive, interactive experience. Powered by Three.js and WebGL, the app features a fully rotatable Earth globe with realistic atmospheric effects and dynamic lighting, creating a captivating visual centerpiece. I engineered smooth, responsive animations and a custom particle trail effect that enhances user engagement, while ensuring flawless performance across all devices.

Marble Racing Game

Created an engaging 3D Marble Game using Three.js, featuring smooth physics-based controls and dynamic environments. Players navigate a marble through challenging tracks with realistic rolling and collision effects, showcasing proficiency in 3D interaction, game mechanics, and responsive design.

Celestial Swirl✨

An immersive, interactive 3D galaxy visualization featuring over 1,000,000 stars rendered procedurally for breathtaking scale and detail. This project uses custom GLSL vertex and fragment shaders to create dynamic star glows, colors, and swirling cosmic patterns with high performance. Users can explore and tweak galaxy parameters in real-time via a sleek GUI, enhancing understanding of particle systems and shader programming. Ideal as a learning tool for advanced graphics concepts or as a stunning visual element in web-based space-themed applications.

The Haunting Grounds

Built a spooky environment Rendered as a fully interactive 3D haunted scene, featuring a detailed haunted mansion with textured walls, roof, door, and surrounding graves. Real-time dynamic lighting and moving ghost lights enhance the eerie atmosphere. Includes realistic shadows, fog, and ambient horror sound to deliver a chilling experience. Smooth camera controls allow exploration of this haunted scene from every angle.
